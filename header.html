Multilingual Header with RTL/LTR Support

This responsive header features a logo that changes position based on the text direction (LTR for English, RTL for Arabic). The layout automatically adjusts when you switch between languages.

Demo Instructions: Click the language buttons (EN/AR) in the header to see the layout change between LTR and RTL modes. The logo will move to the opposite side when switching to RTL.

Left-to-Right (LTR)

Used for languages like English, Spanish, French, etc.

In LTR mode, the logo appears on the left side, navigation in the center, and language switcher on the right.

Right-to-Left (RTL)

Used for languages like Arabic, Hebrew, Persian, etc.

In RTL mode, the logo moves to the right side, navigation remains centered, and language switcher moves to the left.

How It Works

The header uses CSS attribute selectors to change the layout based on the dir attribute of the HTML document:

/* LTR specific styles */
[dir="ltr"] .logo {
  order: 1;
  margin-right: auto;
}

/* RTL specific styles */
[dir="rtl"] .logo {
  order: 4;
  margin-left: auto;
}

When you click the language buttons, JavaScript changes the dir attribute of the HTML document, and the CSS automatically adjusts the layout.
